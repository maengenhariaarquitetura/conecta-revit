# ConectaRevit

Plugin que liga o **Autodesk Revit** ao **Claude Desktop** via MCP, permitindo que o Claude execute operações na API do Revit usando os créditos da sua assinatura.

## Requisitos

- Revit 2025 ou 2026 (Windows)
- Claude Desktop com assinatura Pro ou Max
- .NET 8
